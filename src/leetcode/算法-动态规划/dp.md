动态规划
  主要用于求极值，每一步的状态都是前一步推导而来
  走迷宫例子(走的数字和最小)
      2   100
    1 10  2
      6   90
    每走一步都要保存一个不同状态的最优解 最终得出最优解
  公式(斐波那契例子):
    // 1. 转移方程
    //   1. 定义状态: dp[i]
    //   2. 状态转移方程式: dp[i] = dp[i-1]+dp[i-2]
    // 2. 初始化条件
    // 3. 特殊处理
    function fibByDp(n: number): number {
      const dp = [0, 1];
      for (let i = 2; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2];
      }
      return dp[n];
    }
